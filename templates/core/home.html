Project Overview

AutoStorage is developed using Django for the backend and a separate Raspberry Pi application for interaction with storage robots. This system streamlines warehouse operations by automating inventory management, order processing, and robotic tasks.

Automated Inventory Management

Efficiently manage stock levels and warehouse sections.

Order Processing

Streamline order fulfillment with integrated systems.

Robotic Integration

Control and monitor warehouse robots via Raspberry Pi.

Technologies Used

Django - Backend Web Framework

Django REST Framework - Building RESTful APIs

Python 3.x - Core Language

Installation and Configuration
